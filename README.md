# AutoPhoneRespond
This android project is an automatique responder to a call by answering with DTMF

## What the application is trying to do
* Configuration Activity: 
  * Provide a phone number
  * Provide a specific number for DTMF response
* Core Activity:
  * Off hook a call from the from number configured
  * Using DTMF number to respond to the phone call
  * Hang up the call
